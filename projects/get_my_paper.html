Get My Paper

GetMyPaper is a Python script designed to search for academic papers based on keywords and download the PDF files from various websites, including academic.oup.com, link.springer.com, onlinelibrary.wiley.com, and others. The script retrieves papers matching the search criteria and saves them locally. Each downloaded file contains information such as author names, website/journal name, publication year, and paper title. See Snapshot

Key Features

Keyword Search

Enables users to input keywords for their search, which the script uses to search across multiple websites simultaneously.

Website Selection

Allows users to choose the specific websites they want to search for papers, offering flexibility and customization.

PDF Download

Downloads PDF files from the selected websites, starting from specified pages of the search results. Users can choose to download additional files as needed.

File Structure

Organizes downloaded PDF files with metadata such as author names, website/journal name, publication year, and paper title. This structured approach simplifies the management and referencing of downloaded papers.
